# LCA Design System

## Design Philosophy

Industrial.

Peaceful.

Beautiful.

Readable.

Every object should look engineered.

Never random.

# Shape Language

Use

• Rounded corners

• Circular energy rings

• Large readable silhouettes

Avoid

× Tiny decorations

× Random spikes

× Overly realistic machinery

---

# Motion

Every animation should

Ease In

Ease Out

No sudden stops.

No jitter.

---

# UI Rules

Maximum 4 colors on one screen.

Every important button should feel touchable.

Every panel should have breathing room.

---

# Screenshot Rule

At any moment,

the player should be able to take a screenshot worth sharing.
